# Specification

## Summary
**Goal:** Add a “Schedule today” section on the Daily Dashboard that links users to a provided Google Form.

**Planned changes:**
- Add a clearly labeled “Schedule today” section to the Daily Dashboard page (route “/”).
- Include a visible, clickable link (or button) to the Google Form URL that opens in a new tab with safe external-link attributes.

**User-visible outcome:** On the Daily Dashboard, users can click “Schedule today” to open the scheduling Google Form in a new browser tab.
